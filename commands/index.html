Main Commands

  • Play a song.

    Aliases: p
    Example: play Despacito

  • Check What is Playing.

    Aliases: nowplaying , np
    Bot Permissions: Embed Links

  • Queue System - Shows all currently enqueued songs

    Aliases: queue, q
    Bot Permissions: Embed Links

  • Bassboost - Changes the bassboost settings on the player.

    Aliases: bb
    Bot Permissions: Embed Links

  • Clear - Clear every track from the queue.

    Aliases: clear
    Bot Permissions: Embed Links

  • Disconnect - top the music and leave the voice channel

    Aliases: leave,exit,quit,dc,stop
    Bot Permissions: Embed Links

  • Config - Edit the bot settings

    Aliases: conf
    Bot Permissions: Embed Links

  • Grab - Saves the current song to your Direct Messages

    Aliases: grab, save
    Bot Permissions: Embed Links

  • Help - Information about the bot

    Aliases: -command,commands,cmd [command]
    Bot Permissions: Embed Links

  • Invite - To invite me to your server

    Aliases: inv
    Bot Permissions: Embed Links

  • Loop - Loop the current song

    Aliases: loop, l
    Bot Permissions: Embed Links

  • Loopqueue - Loop the whole queue

    Aliases: Loopqueue, lq
    Bot Permissions: Embed Links

  • Lyrics - Shows the lyrics of the song searched

    Aliases: ly [Song Name]
    Bot Permissions: Embed Links

  • Move - Moves a track to a specified position.

    Aliases: move
    Bot Permissions: Embed Links

  • Pause - Pauses the music

    Aliases: pause, pa
    Bot Permissions: Embed Links

  • Remove - Remove a song from the queue

    Aliases: remove
    Bot Permissions: Embed Links

  • Search - Shows a result of songs based on the search query

    Aliases: search, se
    Bot Permissions: Embed Links

  • Seek - Seek to a position in the song

    Aliases: seek
    Bot Permissions: Embed Links

  • Shuffle - Shuffle the entire queue.

    Aliases: shuffle, shuff
    Bot Permissions: Embed Links

  • Skip - Skip tracks that you don't like!

    Aliases: s, n, next
    Bot Permissions: Embed Links

  • Skipto - Skip to a specific track in queue!

    Aliases: skipto, st
    Bot Permissions: Embed Links

  • Stats - Get information about the bot

    Aliases: stats, about, info
    Bot Permissions: Embed Links

  • Volume - Check or change the current volume

    Aliases: vol, v
    Bot Permissions: Embed Links

  • Ping - Get ping information about the bot

    Bot Permissions: Embed Links
